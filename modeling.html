Modeling

Algonquin College A Building

This project was done as part of one of our classes as groups. The main purpose of this project was to give the ability to transform a 2D CAD drawing into a 3D BIM model and increase the ability to collaborate on a project.
My group's section of the building has 2 storeys above grade and a basement. There are three entrances and staircases. On the ground floor, there is a Beauty and Spa Salon that covers coiffeur, mani-pedi, facial care and massage rooms. There are classrooms and offices on the second floor.
For modeling, AutoCAD files and Point Cloud file that was attached to Revit was used as references. The point cloud was showing that the building was located slightly different than the CAD drawings. Thus, the model was placed in line with point cloud. The collaboration was done through BIM 360 and worksets. The levels in Revit file wasn’t matching the AutoCAD levels. So, we adjusted the levels.
I was responsible of the foundations, superstructure and floors.

Family Parameters

It is very important to be able to customize the family parameters. But this can be very time consuming process. Using Dynamo, it's easy to automate this process.
For the school assignment, I created custom nodes for this purpose that can be applied for any project. With these nodes it's easy to pull family parameters, list them and export these lists to an excel file. Or change the values of existing family parameters by pushing the data in an excel file to Revit.
